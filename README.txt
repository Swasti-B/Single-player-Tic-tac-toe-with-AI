File included: TictacToe.py

This is an unbeatable single player Tic tac toe game. Program is built in python using the Mininmax algorithm. 
X- Human player
O- AI
For every move made by the user, AI checks for all the possible move in that given state and makes the most appropriate move that could lead AI to have maximum payoff. For an optimal player, the maximum payoff is having a tie while, with a non-optimal player, AI wins the game mostly. AI takes the move/path that gives the maximum utility.
